🔥 Connexion à Firebase

Entre les informations de ton projet Firebase pour que toutes les données soient partagées en temps réel sur tous les appareils.

1
Va sur console.firebase.google.com → crée un projet → active Realtime Database (mode test)
2
Dans les paramètres du projet → "Ajouter une application Web" → copie les infos ci-dessous
3
Dans Realtime Database → Rules → remplace par {"rules":{".read":true,".write":true}}
Tu n'as besoin de faire ça qu'une seule fois — les infos seront sauvegardées.
